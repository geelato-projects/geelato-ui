### Lints and fixes files
```
npm run lint
```


### About ESLint
ESLint: Unexpected console statement. (no-console)
```
{
    "rules":{
        "no-console":0
    }
}
```
ESLint: 'reject' is defined but never used. (no-unused-vars)
```
"no-unused-vars": 'off'
```
